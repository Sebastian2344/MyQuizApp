# Quiz
Aplikacja umożliwia stworzenie quizu i rozegranie go. 
Posiada ona bazę z pytaniami która jest do wypełnienia przez użytkownika. 
Pytania można usuwać, dodawać i edytować.
### Bibloteki wykorzystane w aplikacji:
  `sqflite`,
  `path_provider`,
  `path`,
  `bloc`,
  `flutter_bloc`,
  `freezed_annotation`,
  `go_router`,
  `equatable`,
  `cupertino_icons`,
  `bloc_test`,
  `mocktail`.
